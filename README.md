# Cognifyz Technologies - Zomato Data Analysis Internship Project

## Overview
This project analyzes the Zomato Restaurant Dataset containing 9,551 restaurants. It was completed as part of the Cognifyz Technologies Data Analysis Internship and covers both Level 1 and Level 2 tasks.

## Dataset
- **Source:** `zomato.csv`
- **Total restaurants:** 9,551
- **Focus:** restaurant cuisines, city distribution, pricing, delivery options, ratings, cuisine combinations, geographic mapping, and chains.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium

## Level 1 Tasks
### Task 1: Top 3 Cuisines
- North Indian: **41.5%**
- Chinese: **28.66%**
- Fast Food: **20.81%**

### Task 2: City Analysis
- **New Delhi** has the most restaurants.
- **London** has the highest average rating at **4.54 ⭐**.

### Task 3: Price Range Analysis
- Budget: **46.51%**
- Moderate: **32.62%**
- Expensive: **14.72%**
- Luxury: **6.14%**

### Task 4: Online Delivery Analysis
- **25.69%** of restaurants offer online delivery.
- Restaurants with delivery have a higher average rating: **3.249** vs **2.464**.

## Level 2 Tasks
### Task 1: Restaurant Ratings Distribution
- Analyzed how ratings are distributed across restaurant listings.

### Task 2: Cuisine Combinations
- Investigated the most common cuisine combinations offered by restaurants.

### Task 3: Geographic Mapping with Folium
- Created interactive maps to visualize restaurant locations.

### Task 4: Restaurant Chains Analysis
- Identified popular restaurant chains and their presence in the dataset.

## Key Findings
- North Indian, Chinese, and Fast Food dominate the dataset.
- New Delhi is the largest city by restaurant count, while London leads in quality by rating.
- Most restaurants are budget-friendly, with fewer listings in the luxury segment.
- Online delivery correlates with higher ratings.
- Advanced analysis reveals rating distribution patterns, cuisine pairings, geographic location clusters, and chain prevalence.

## Project Structure
